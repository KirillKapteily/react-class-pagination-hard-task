# React Class Pagination Refactor

Це навчальний React-проєкт на **class components**.

Твоє завдання: переписати його на **function components + hooks**.

## Що є в проєкті

- завантаження posts/users з API
- пагінація
- пошук
- фільтр по користувачу
- сортування
- вибір кількості постів на сторінці
- localStorage для `perPage`
- loading/error/empty states

## Запуск

```bash
npm install
npm run dev
```
